# چالش های سمت سرور

در ساخت سیستم سمت سرور نسبت به کاری می خواهیم انجام بدهیم چالش وجود دارد ، سیستمهای مدیریت بانک اطلاعاتی ده ها سال هست وجود دارند ، اکنون این سیستم های بصورت گسترده در حال استفاده در برنامه های تحت وب می باشد معمولا ارتباط سرور به صورت REST/CRUD می باشد

پیاده سازی کامل CRUD در هسته سیستم انجام شد ، این موضوع امکان بهره موثر از حافظه کش سمت مرورگر فراهم کرده است

# نیازمندی ها

  • توسعه سریع
  • استفاده از زبان TypeScript برای دامین سمت سرور
  • استفاده موثر از async functions
  • داشتن data-layer منسجم
  • ایجاد Code-Generator
  • استفاده موثر از REST/CRUD
  • تنظیمات پیشرفته

# چالش های متناسب با نیازمندی ها

  • قابلیت منحصر به فرد Hot Reload در سمت سرور
  • تنظیم مناسب webpack برای تولید کد این بار در سمت سرور و برای Nodejs
  • توسعه ExpressJS برای امکان بهره مندی از قابلیت Promiseو Async
  • لود فایل های markdown برای لایه دسترسی به داده
  • تولید سیستم کدساز به صورت چندتیکه
  • پیاده سازی کامل CRUD
  • انجام تنظیمات نسبت محیط توسعه یا عملیاتی

# نحوه عملکرد سرور

دیداصلی این سرور برای سرویس های JSON بوده است امکاناتی (فرعی) به شما می دهند که امکان هندل فایل های استاتیک نیز می دهند قابلیت فایل سرور برای سهولت استفاده است ، لطفا برای مقیاس های بزرگتر از nginx استفاده کنید

پیشوند موجود در سمت وب سرور

  • /api : توابع و پروسه های کار با اطلاعات سیستم
  • /assets : فایل سرور
  • /login : ماژول لاگین
