# فولدر examples

این فولدر محل قرار گیری دامین سمت سرور می باشد

# فولدر examples/sample1

کد سپیداران در اینجا قرار دارد ، جهت اطلاعات بیشتر به سند کد های دامین مراجعه فرمایید

# فایل package.json

این فایل در همه پروژه های Nodejs وجود دارد و مشخص می کند که از کدام ماژول ها استفاده شده است

# فولدر src

این فولدر شامل هسته سمت سرور می باشد که کد های دامین در این فولدر نیست

# src/code-gen

سیستم تولید کد با کاربرد های مختلف

  • DataAccess Layer
  • Database - DDL
  • TypeScript Interface DTO
  • Frontend - Typescript React - به صورت کمکی

وجود این فولدر در هنگام اجرا در حال production ضرورتی ندارد

# src/core

هسته سمت سرور در این فولدر می باشد

  • توابع جهت ایجاد API - امکان خروجی Async می دهد
  • ارتباط با بانک اطلاعاتی
  • کلاس های لایه دیتابیس
  • گزارش گیری
  • مدیریت موثر سیستم کش سمت مرورگر در هنگام خواندن اطلاعات از دیتابیس
  • پیاده سازی موثر CRUD
  • پیاده سازی موثر Master/Detial

# src/dts

فایل های توصیفی جهت دامین و ایجاد ماژول برای توسعه به زبان TypeScript

این فولدر با نام types نیز در شناخته می شود namespaces-and-modules
