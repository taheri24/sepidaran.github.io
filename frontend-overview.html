# Deployment

this application run on windows-server by nssm.exe utility in our server.

bundle:

  • base.js --> external link of organic-ui(our-framework)
  • vendors --> react,react-dom and other node_modules
  • domain.js --> result of built of domain by domain-developer generated via webpack
  • domain_FA-IR.js --> resource file for application localzation(Farsi-IRAN)
  • domain_EN-US.js -->resource file for application localzation(English-United States)

# Tip

for increase speed development , we desire that build base.js&vendors.js have another build-process & anthoer webpack configuaration, domain-developer use prepared bundles(base.js&vendors.js) for development . domain-developer just build domain*.js files .
