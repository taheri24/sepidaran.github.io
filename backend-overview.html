# Deployment

this application run on windows-server by nssm.exe utility in our server.

contents:

  • nssm.exe --> Windows Service Provider (run console application as windows-service)
  • node.exe --> single-executable nodejs application
  • database-connection.json --> Database Connection Properties
  • src(folder) --> sepid-json-api-server
  • dist(folder) --> bundle of domain
REM install-sepidaran.cmd 
setlocal
cd /d %~dp0
 
SET CURRENTDIR=%cd%
SET nssm=%CURRENTDIR%\nssm.exe 
SET SERVICE_NAME=sepidaran_service
%nssm% stop %SERVICE_NAME% confirm
TIMEOUT   3

%nssm% remove %SERVICE_NAME% confirm

TIMEOUT   3
%nssm% install  %SERVICE_NAME% "%CURRENTDIR%\node.exe" "%CURRENTDIR%\src\index.js" --databaseConnection="%CURRENTDIR%\database-connection.json"   --serverPort=8081 
%nssm% set %SERVICE_NAME% AppDirectory "%CURRENTDIR%"
%nssm%  start  %SERVICE_NAME%
